Тема: Программирование в ограничениях

Составление программы тренировок
(что-то похожее на составление расписания)

Суть программы
Программа составляет план тренировок на неделю по предпочтениям пользователя 
(формирует тренировочные дни путем распределения упражнений между ними)

Файл exercises.py
Содержит набор упражнений и связанные с ними группы мышц. Также есть кардио упражнения.

Файл input.py
Пользователь задает минимальное и максимальное допустимое количество упражнений в тренировочный день.
Затем пользователь задает количество упражнений на КАЖДУЮ группу мышц в неделю по отдельности.

Условности
Программа распределят упражнения согласно насройкам пользователя - соблюсти минимальное и максимальное количество упражнений в тренировочный день, вывести то количество упражнений на определенную группу мышц, которое задал пользователь (не больше и не меньше того).
Не может быть дня с лишь одним упражнением, по возможности это упражнение перекидывается на другой занятый день.
Если после распределения получается день с количеством упражнений меньше минимального, то в список упражнений в этот день добавляется ровно ОДНО КАРДИО-упражнение.
