Тема: Программирование в ограничениях (с использованием генетического программирования)

Составление программы тренировок
//* За основу взят реализованный ранее алгоритм (папка Old verion) - та же суть, но с использованием метода случайного выбора *\\

Суть программы
Программа составляет план тренировок на неделю по предпочтениям пользователя 
(формирует тренировочные дни путем распределения упражнений между ними)

Файл exercises.py
Содержит набор упражнений и связанные с ними группы мышц. Также есть кардио упражнения.

Файл input.py (ВХОДНЫЕ ДАННЫЕ)
Пользователь задает минимальное и максимальное допустимое количество упражнений в тренировочный день.
Затем пользователь задает количество упражнений на КАЖДУЮ группу мышц в неделю по отдельности.
Задаются значения для численности популяции и количества поколений скрещивания и селекции.

Этапы генетического программирования:
1. Инициализация популяции: Создаем начальную популяцию распределений
2. Оценка пригодности: Оцениваем каждое распределение по некоторой функции пригодности
3. Селекция: Выбираем лучшие распределения для создания новой популяции
4. Скрещивание: Комбинируем части распределений, чтобы создать новое распределение
5. Мутация: Вносим случайные изменения в новое распределение для поддержания разнообразия
6. Итерация: Повторяем процесс до тех пор, пока не достигнем приемлемого решения
