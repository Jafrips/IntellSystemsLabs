Тема: Программирование в ограничениях (с использованием метода случайного выбора)

Составление программы тренировок

*** Суть программы и алгоритм ***
Программа составляет план тренировок на неделю по предпочтениям пользователя 
(формирует тренировочные дни путем распределения упражнений между ними)
***
Для каждой группы мышц и заданного количества упражнений выбирает случайное упражнение и случайный день для него.
Проверяет каждый день на соответствие минимальному количеству упражнений. Если в дне меньше упражнений, добавляет кардио-упражнение.
Использование метода случайного выбора позволяет каждый раз получать отличный от предыдущего результат, но всегда будет удовлетворять заданным ограничениям.
**********************************

Файл exercises.py
Содержит набор упражнений и связанные с ними группы мышц. Также есть кардио упражнения.

Файл input.py
Пользователь задает минимальное и максимальное допустимое количество упражнений в тренировочный день.
Затем пользователь задает количество упражнений на КАЖДУЮ группу мышц в неделю по отдельности.
